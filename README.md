# 2.MCP是什么

Model Context Protocol（模型上下文协议），是一个开放的协议标准，定义了AI模型与外部工具、数据源、服务等进行交互的方式和规范。通过MCP，AI模型可以访问和利用各种外部资源，增强其功能和应用场景。
